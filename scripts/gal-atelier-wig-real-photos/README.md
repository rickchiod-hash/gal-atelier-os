# Gal Atelier — fotos reais para segmento wigmaker

Pacote com 9 imagens reais/stock de wigs, laces, perucas em manequins, vitrine e textura de cabelo para substituir as ilustrações provisórias do projeto Gal Atelier.

## Como usar

- Copie a pasta `images/` para `public/images/wigs/` ou para o diretório de assets do frontend.
- Use imagens reais no hero, catálogo, sobre, stories e cards de serviço.
- Para produção, recomendo converter para WebP/AVIF no pipeline do projeto ou deixar o componente de imagem do framework otimizar.
