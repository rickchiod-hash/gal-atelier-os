# Photo Review Checklist — Produto completo

Use este checklist para cada produto do catálogo.

## Obrigatório por produto

- [ ] `hero.png` — imagem principal limpa, premium e desejável.
- [ ] `lace-close.png` — macro da linha frontal/lace/hairline quando aplicável.
- [ ] `inside-cap.png` — construção interna: cap, elásticos, pentes, costuras.
- [ ] `texture.png` — textura do cabelo em close.
- [ ] `side-view.png` — visão lateral para volume e caimento.
- [ ] `scale-length.png` — referência de comprimento.
- [ ] `before-after.png` — obrigatório para serviços de manutenção, reforma, coloração e higienização.
- [ ] `educational.png` — imagem com labels discretos para itens técnicos.
- [ ] Alt text claro, sem keyword stuffing.
- [ ] Nome do arquivo em kebab-case.
- [ ] Fundo consistente com a paleta LaceLab.

## Critério visual

- Fundo: Warm Ivory / Porcelain.
- Acentos: Rosewood, Deep Espresso, Champagne Gold.
- Evitar azul saturado.
- Evitar poluição visual.
- Evitar excesso de texto dentro da imagem.
- Priorizar textura, hairline e acabamento.

## Reprovação automática

- Foto com baixa nitidez.
- Foto sem foco no produto.
- Imagem com marca d'água de terceiros.
- Imagem sem licença/autorização.
- Foto que não mostra detalhe suficiente para decisão de compra.
- Arte genérica que parece banco de imagem aleatório.
