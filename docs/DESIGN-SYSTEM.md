# Design System V5

## Princípios

- Clean primeiro.
- Contraste alto.
- Menos gradiente, mais hierarquia.
- Espaçamento fluido com clamp.
- Mobile-first.
- Estados visíveis.
- Sem layout quebrando em telas pequenas.

## Tokens

- Cores: `--color-*`
- Espaços: `--space-*`
- Tipografia: `--font-size-*`
- Raios: `--radius-*`
- Sombras: `--shadow-*`

## Regra

Não criar CSS aleatório fora dos tokens sem justificar.
